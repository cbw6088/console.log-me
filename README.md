# Blio

이력서, 포트폴리오, 블로그를 하나로 관리하고 작성한 모든 활동을 GitHub 잔디로 기록하는 프로젝트입니다.

## 프로젝트 소개

Blio는 개발자들의 커리어 관리를 위한 올인원 플랫폼입니다. 블로그 글 작성, 포트폴리오 제작, 자기소개서 작성을 지원하며, 모든 활동이 GitHub 커밋으로 자동 기록되어 꾸준한 성장을 증명할 수 있습니다.

_(프로젝트 설명은 추후 업데이트 예정입니다)_

## 기술 스택

### Core
- **Next.js** 15.5.0 - React 프레임워크
- **React** 19.1.0 - UI 라이브러리
- **TypeScript** 5 - 타입 안정성

### Styling
- **Tailwind CSS** 4 - 유틸리티 기반 CSS 프레임워크
- **Tailwind CSS Typography** - 타이포그래피 플러그인
- **Lucide React** - 아이콘 라이브러리

### Editor
- **Tiptap** - 리치 텍스트 에디터
  - `@tiptap/react`
  - `@tiptap/starter-kit`
  - `@tiptap/extension-image`
  - `@tiptap/extension-placeholder`
  - `@tiptap/extension-text-style`

### Utilities
- **react-resizable** - 리사이징 가능한 컴포넌트
- **classnames** - 조건부 className 유틸리티

### Development Tools
- **ESLint** - 코드 품질 관리
- **Turbopack** - 빠른 번들링 및 개발 서버
